# 🎨 Asimov

**Ferramentas de IA:** ChatGPT, Gemini, GitHub Copilot e Figma AI.

**Etapas principais:** IA foi essencial na estruturação de componentes, implementação de responsividade, criação de animações e identificação de bugs.

**Ajustes manuais:** Consistência visual com o design, refinamento de espaçamento/tipografia/cores e simplificação de soluções complexas para evitar desnecessária complexidade.

Todas as decisões finais foram validadas manualmente para garantir **qualidade** e **usabilidade**.
